== GameRate

Project I created after GA WDI. A yelp clone for venues that have board games.

[Visit GameRate Here](https://fast-escarpment-3511.herokuapp.com/)

Technologies I Used For This Project:

* Ruby on Rails

* jQuery

* Bootstrap

* PostgreSQL

* Google Maps API

* Amazon S3

* ElasticSearch

* CarrierWave 

* Fog 

* Figaro 

* Mini_Magick

* Devise

* SearchKick

* jQuery Raty

* ImageMagick
